## Scripts for the monitoring program of insects in hollow oaks

This is a collection of mostly R scripts related to the monitoring program of insects in hollow oaks, see https://www.nina.no/Naturmangfold/Insekter/Overv%C3%A5king-av-insekter-i-hule-eiker. It documents the selection procedure of which trees to include in the monitoring program together with some analysis code.

The scripts are published "as is", and are not polished or intended to be read by a general audience.

Data that are referred to in the reports, such as lists of localities, are found under season20xx/reportData.
